# Telegram Bot Gemini

### Quick Start
## 1. Clone project to local

```bash
git https://github.com/Bubble-droid/Telegram-Bot-Gemini.git
cd Telegram-Bot-Gemini
```

## 2. Install dependencies

```bash
npm install
```
